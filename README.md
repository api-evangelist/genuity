# Genuity (genuity)

Genuity Inc. was an internet infrastructure services company spun off from GTE/BBN, which filed for bankruptcy in 2002 and had the majority of its assets acquired by Level 3 Communications. Level 3 was later acquired by CenturyLink in 2017, which rebranded as Lumen Technologies. This repository tracks the legacy Genuity entity and its successor companies.
